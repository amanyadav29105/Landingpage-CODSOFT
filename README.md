# Landingpage🌍 TravelX - Professional Travel Agency Website

A modern, responsive travel website built using HTML, Tailwind CSS, and JavaScript with smooth animations and a working contact form powered by Formspree.

---

🚀 Live Features

- ✨ Modern UI with Tailwind CSS
- 📱 Fully Responsive Design
- 🎬 Smooth Animations using AOS
- 📩 Working Contact Form (Formspree)
- 📌 Scroll-to-top button
- 📂 Clean Folder Structure

---

📁 Project Structure

TravelX/
│── index.html
│── css/
│   └── style.css
│── js/
│   └── script.js

---

🛠️ Technologies Used

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Vanilla JS)
- AOS (Animate On Scroll)
- Formspree (for contact form)

---

📩 Contact Form Integration

This project uses Formspree for handling form submissions.

Form Endpoint Used:

https://formspree.io/f/mykarwbp

Features:

- Sends form data directly to your email
- No backend required
- Spam protection using hidden field (_gotcha)
